🎩 mágico

Animaciones CSS3 con efectos especiales. MANIFESTACIÓN

Instalación

NPM - url del paquete

npm i magic.css

YARN - Paquete url

hilo añadir magic.css

Inclusión

Incluya el archivo magic.css o incluya la versión minified magic.min.css

< link  rel = " hoja de estilo "  href = " yourpath / magic.css " >

o

< link  rel = " hoja de estilo "  href = " yourpath / magic.min.css " >

Uso con JavaScript

Este es un código de ejemplo en Hover con JavaScript , primero debes incluir la clase "tiempo mágico" y después de la animación deseada.

Uso con jQuery

Este es un código de ejemplo en hover con jQuery, primero debes incluir la clase "tiempo mágico" y luego la animación que deseas.

$ ( ' .yourdiv ' ). hover ( function () {
     $ ( this ). addClass ( ' magictime puffIn ' );
});

Si desea cargar la animación después de cierto tiempo, puede usar este ejemplo:

// configura el temporizador a 5 segundos, después de eso, carga la animación mágica 
setTimeout ( function () {
     $ ( ' .yourdiv ' ). addClass ( ' magictime puffIn ' );
}, 5000 );

Si desea cargar la animación después de cierto tiempo pero con un bucle infinito, puede usar este ejemplo:

// configura el temporizador a 3 segundos, después de eso, carga la animación mágica y repite para siempre 
setInterval ( function () {
     $ ( ' .yourdiv ' ). toggleClass ( ' magictime puffIn ' );
}, 3000 );

Consejos HTML y CSS

Puedes cambiar el tiempo de la animación configurando la clase "tiempo mágico" por ejemplo:

.magictime {
     -webkit-animation-duration : 3 s ;
    animación-duración : 3 s ;
}

El tiempo predeterminado de CSS es:

.magictime {
     -webkit-animation-duration : 1 s ;
    animación-duración : 1 s ;
}

Si desea asignar el tiempo a una animación específica , puede usar el siguiente código (usar clase 2) :

.magictime.magic {
     -webkit-animation-duration : 10 s ;
    animación-duración : 10 s ;
}

🎉 Compilación de Gulp y SCSS (SASS).

Si quieres personalizar los archivos CSS, ahora tendrás la oportunidad. Por ejemplo, si desea incluir solo ciertas animaciones, tendrá que ir a este archivo:

activos / scss / magic.scss

Comente o descomente su archivo deseado y ejecute desde la terminal los siguientes comandos:

npm instalar

y último comando:

trago

Automáticamente esto genera los nuevos archivos!
